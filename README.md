# Project Setup

## Setup Directories

```cmd
cd/you_desired_directory/
mkdir any_name
```

## cloning the project

```bash
    git clone https://github.com/mabdullahadeel/hulu-clone.git
```

## Installing dependencies

```bash
    npm install | npm i | yarn install | yarn
```

## Running Project Locally

```bash
    npm start | yarn start
```
